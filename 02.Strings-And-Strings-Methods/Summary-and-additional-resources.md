#### Summary and Additional Resources
In this chapter, you learned the ins and outs of Python string objects.
You learned how to access different characters in a string using subscripts and slices, as well as how to determine the length of a string
with len().

Strings come with numerous methods. The .upper() and .lower()
methods convert all characters of a string to upper or lower case,
respectively. The .rstrip(), .lstrip(), and strip() methods remove
whitespace from strings, and the .startswith() and .endswith()
methods will tell you if a string starts or ends with a given substring.
You also saw how to capture input from a user as a string using the input() function, and how to convert that input to a number using int()
and float(). To convert numbers, and other objects, to strings, you
use str().

Finally, you saw how the .find() and .replace() methods are used to
find the location of a substring and replace a substring with a new
string.
